# Body Mass Index Calculator

This is a Web application to calculate you body mass index built with Spring Boot


## Prerequisites
* JDK 8+
* Apache Maven 3.5+

## Build Dockerfile
sudo docker build . -t java-app

# Run container
docker container run --name java-app -p 8000:8000 java-app
